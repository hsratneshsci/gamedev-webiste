HIGH-LEVEL PROJECT SUMMARY
This project is made to educate people with rocket launch sequence and life in space This game is aimed for middle schoolers(age : 10 to 15) as audience . This game has engaging story which leads to make the students to learn while they play. This game consist learning content like Launch sequences , launch tests ,things they do before launch, life in space, and what do they do In space missions. Programming language is used in pygame , renpy scripting. This game is multiplatform made this game for android ,windows and Linux. This game has multi-character/protagonist and one forth-wall breaking character. OUR GAMEPLAY IS INCLUDED IN DEMO ALSO IN GITHUB PAGE
